# Christmas Lights Simulator

A festive front-end mini project that simulates a string of animated Christmas lights. You can start and stop the light show, change the flashing speed, and customize the bulb colors with a built-in palette editor.

## Features

- Animated seven-bulb light string
- Start and stop playback controls
- Faster and slower speed controls
- Custom color picker for each bulb
- Responsive layout for desktop and mobile screens

## Project Structure

- `index.html` - page structure and UI sections
- `index.css` - layout, theme, animations, and responsive styling
- `index.js` - light show logic, speed control, and palette behavior

## How to Run

1. Open the project folder.
2. Launch `index.html` in your browser.

No build tools or dependencies are required.

## How to Use

- Click `Start Light Show` to begin the animation.
- Click `Stop Light Show` to stop it.
- Use `Faster` and `Slower` to change the flashing speed.
- Click `Customize Palette` to open the color editor.
- Pick a color for each bulb and click `Apply Palette`.
